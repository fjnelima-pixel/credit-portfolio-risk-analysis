### Credit Portfolio Risk & Performance Analysis

### Objective
This project simulates a retail credit portfolio and applies analytical techniques to evaluate portfolio health, identify risk patterns, and model default probability.

### Key Components

Synthetic loan portfolio generation

Portfolio risk metrics & exposure analysis

Credit risk segmentation

Logistic regression default model

Business-style insights

### Tools Used

Python (Pandas, NumPy)

Scikit-Learn

Data Visualization (Matplotlib)
